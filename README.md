# 📓 Notebook.Me Website

Official website for **Notebook.Me** — a fast, minimal, open-source Java notebook application.

---

## 🚀 About

This repository contains the landing page for Notebook.Me, built using:

* HTML
* CSS
* JavaScript

The website showcases features, download links, and project information.

## ⚙️ Getting Started

1. Clone the repository:

   ```bash
   git clone https://github.com/YOUR-USERNAME/notebook-website.git
   ```

2. Open the folder

3. Run:

   * Double click `index.html`
     OR
   * Use Live Server (recommended)
